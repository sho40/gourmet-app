## サイト構想

InstagramグラフAPIを利用しタグのいいね数を起点に人気の投稿を取得。
食べ物のみ取得できるようフィルタリングする。

## 機能

- 人気グルメの一覧表示
- 詳細画面
- タグをグループ化して、カテゴリごとに表示
- 検索機能
